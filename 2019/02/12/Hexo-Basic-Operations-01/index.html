新建并部署一篇文章

hexo n “博客文章名“ == hexo new “博客文章名“ #新建文章

hexo g == hexo generate#渲染并生成静态页面至public目录

hexo s == hexo server #开启本地预览(默认端口4000，’ctrl + c’关闭server)

hexo d == hexo deploy#部署到服务器(将.deploy目录部署到GitHub等服务器)

hexo new page “pageName” #新建页面

给新建的文章添加分类和标签:

---
title: title #文章标题
date: 2019-02-12 13:47:44 #文章生成时间
categories: Hexo教程 #文章分类目录 可以省略
tags: 标签1, 标签2   #文章标签 可以省略
comments: true/false 
---

服务器

hexo server #Hexo实时监视文件变动并自动更新，无须重启服务器。

hexo clean #清除缓存 一般仅用于本地，在网页正常情况下可以忽略此条命令

hexo server -s #静态模式

hexo server -p 5000 #更改端口，当默认4000端口被占用情况下使用。

hexo server -i 192.168.1.1 #自定义 IP
